### 同源策略
- 是浏览器的安全策略
- 协议名、域名、端口号必须完全一致
### 跨域
- 违背同源策略就会产生跨域
### 解决跨域
- jsonp cors 服务器代理

------------------------------
#### **jsonp解决跨域**
```
// 创建script标签
var script = document.createElement('script')
// 设置回调函数
functiongetData(data){
    // 数据请求回来会被触发的函数
    console.log(data)
}
// 设置script的src属性，设置请求地址
script.src = 'http://localhost:3000?callback=getData'
// 使script生效
document.body.appendChild(script)
```
